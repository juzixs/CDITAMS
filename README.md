# 驰达IT资产管理系统 (CDITAMS)

驰达IT资产管理系统（CDITAMS）是一款面向企业的IT资产全生命周期管理软件，帮助企业实现IT资产的规范化、信息化管理。

## 功能特性

### 资产管理
- 设备管理（增删改查、分配、报障、报废）
- 资产分类管理（支持1-4级分类）
- 位置管理（支持1-4级层级：园区→设施→楼层→区域）
- 设备地图可视化（楼层平面图绘制）
- 工位管理（工位绑定设备）
- 软件管理（软件资产、授权分配、字段自定义）
- 耗材管理
- 服务管理（服务请求、服务合同）
- 服务合同管理（服务类型、价格、合同期限）

### 移动端优化
- 设备列表卡片视图（自定义显示字段）
- 软件列表卡片视图（自定义显示字段）
- 响应式设计适配移动端

### 组织管理
- 用户管理
- 部门管理（树形结构）
- 角色权限管理

### 盘点管理
- 盘点计划
- 盘点任务
- 盘点执行
- 盘点报表

### 设备地图功能（基于 Konva.js）

#### 图层架构（4 层）

| 图层 | 内容 |
|------|------|
| Background Layer | 底图、网格 |
| Element Layer | 画线、虚线、矩形、门 |
| Object Layer | 工位、区域、文字 |
| UI Layer | 选择框、临时绘制图形 |

#### 地图查看器
- 自适应缩放，地图完整显示在画布内
- 拖拽平移查看完整地图
- 点击工位打开右侧详情面板
- 详情面板内搜索绑定/解绑设备
- 工位状态颜色区分（已占用/维修中/可用）
- 设备状态圆点显示（正常/故障/维修/闲置）
- 未绑定设备列表固定显示在右下角
- 窗口缩放时自动重新适配地图

#### 设备详情页
- 位置信息下方显示平面图缩略图
- 所绑定工位在缩略图中红色高亮

## 技术栈

- **后端**: Django 5.2+
- **前端**: HTML5, CSS3, JavaScript, Konva.js 9.x
- **数据库**: SQLite（默认）/ MySQL/PostgreSQL
- **图形渲染**: Konva.js（Canvas 封装库）

## 快速开始

### 1. 环境要求

- Python 3.8+
- Django 5.2+

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 数据库初始化

```bash
# 应用数据库迁移
python manage.py migrate

# 使用备份数据库（含默认数据和超级管理员）
cp db.sqlite3.bak db.sqlite3
python manage.py migrate

# 初始化默认数据（仅空数据库需要）
# 包含：权限、角色、部门、资产分类、位置、服务类型、系统配置、设备字段、软件字段等
python init_data.py
```

### 4. 启动服务

```bash
python manage.py runserver 0.0.0.0:8000
```

### 5. 访问系统

- 系统地址: http://127.0.0.1:8000/
- 管理后台: http://127.0.0.1:8000/admin/

## 设备地图使用流程

### 位置层级说明

| 级别 | 名称 | 示例 | 说明 |
|------|------|------|------|
| 1级 | 园区 | 产业园区、草滩园区 | 最高层级 |
| 2级 | 设施 | 办公楼、生产车间 | 园区内的建筑物，设置楼层数后自动生成楼层 |
| 3级 | 楼层 | 1楼、2楼、3楼 | 绘制平面图的位置 |
| 4级 | 区域 | A201室、办公区 | 绑定到3级楼层的平面图内 |

### 步骤1：创建位置层级

1. 进入 **资产管理 → 位置管理**
2. 依次创建：
   - **1级 - 园区**（如：产业园区，编码：CY）
   - **2级 - 设施**（如：办公楼，编码：CYOA），设置**楼层数**（如：7）
   - 系统会自动生成3级楼层位置（如：B1层、1楼、2楼...5楼、天台）

### 步骤2：启用平面图

1. 编辑3级位置（楼层），如"2楼"
2. 勾选"启用平面图"
3. 设置画布宽度/高度（如：1200×800）
4. 保存

### 步骤3：编辑平面图

1. 点击"编辑地图"按钮
2. 使用工具栏绘制（每按钮有详细提示，鼠标悬停可见）
3. 支持上传底图作为参考（不保存）
4. 右侧图层面板控制各图层显示/隐藏
5. 点击"保存"

### 步骤4：查看地图 & 绑定设备

1. 点击"设备地图"进入地图查看页面
2. 地图自适应缩放，完整显示在画布内
3. 点击工位，右侧弹出详情面板
4. 搜索并绑定设备

### 步骤5：设备详情查看地图

1. 进入设备详情页
2. 如果设备绑定了工位，右侧会显示平面图缩略图
3. 缩略图中绑定的工位以红色高亮显示

## RESTful API

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/assets/api/map-data/<location_id>/` | 获取地图数据 |
| POST | `/assets/locations/elements/save/` | 保存地图元素 |
| GET | `/assets/api/devices/search/?q=<query>` | 搜索设备 |
| POST | `/assets/api/workstations/<ws_id>/bind-device/` | 绑定设备到工位 |
| POST | `/assets/api/workstations/<ws_id>/unbind-device/` | 解绑设备 |
| GET | `/assets/api/location-tree/` | 获取位置树 |
